# Workspace Rules — Scientific Diagram Editing

This file defines the project-scoped rules and style guidelines for agents working in this workspace.

## Agent Guidelines & Protocols

- **Focus Area**: Scientific Diagram Editing and Revision Analysis.
- **Language**: Traditional Chinese (繁體中文) for analysis reports, English for codebase development and skill configuration.
- **Output Preferences**: Maintain highly clean, readable diagrams. Prefer vectorized SVG operations.
- **Preservation**: Do not discard metadata or layout rules when updating SVGs. Preserve coordinates where possible.
- **Verification Rule**: Verify all edited figures visually or structurally before declaring a task done.
